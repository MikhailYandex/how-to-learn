# Научиться учиться. Веб-страница для Яндекс.Практикума
------

Простая, неадаптивная веб-страница с анимациями.

## Реализация
* Flexbox верстка
* CSS-анимация
* BEM-методология
* Файловая структура проекта по правилам Nested БЭМ
